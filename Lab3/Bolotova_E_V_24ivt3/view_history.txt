ответы на вопросы:
1. Как посмотреть изменения в конкретном коммите?
git show <хэш_коммита>
2. Как найти коммит по сообщению?
git log --grep="текст_сообщения"
3. Как выполнялось решение конфликта?
Путем редактирования файлов с конфликтами, затем выполняется git add для помеченных файлов и git commit для завершения разрешения конфликта.
